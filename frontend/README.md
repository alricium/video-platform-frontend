# Video Platform Frontend

Bu proje, tamamen **frontend tabanlı bir video platformu** sağlar.  
Kullanıcılar kayıt olabilir, giriş yapabilir, videolar yükleyebilir, videoları görüntüleyebilir, beğenebilir ve yorum yapabilir.

Proje React + TypeScript + Vite ile geliştirilmiştir.

---

## Özellikler

- Kullanıcı kayıt ve giriş sistemi (frontend taraflı)
- Video yükleme ve yönetimi
- Videoların Home sayfasında listelenmesi
- Video beğenme (like) özelliği
- Yorum yapma ve görüntüleme
- Modern React + TypeScript + Vite altyapısı
- HMR (Hot Module Replacement) ve hızlı geliştirme

---
